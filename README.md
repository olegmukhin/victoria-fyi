# Highway Boss Run

Small browser game prototype built with plain HTML, CSS, and JavaScript.

## Play

Open [index.html](./index.html) in a browser, or serve the folder with a local static server.

## Controls

- `Left` / `A`: move left
- `Right` / `D`: move right
- `G`: open the garage
- `Enter` / `Space`: start or restart

## Goal

Survive the highway section, reach the boss rig, then stay in its lane long enough to break its armor while dodging its attacks.

## Progression

- You start with one random animal driver out of 40 total.
- Beating a boss unlocks one random new driver.
- Rainbow drops during boss fights unlock one of 20 paint colors.
- Use the garage menu to choose from the drivers and colors you have collected.
